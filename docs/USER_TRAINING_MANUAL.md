# Task Reminder User Training Manual

## Overview

Task Reminder is the office command center for front desk workflows:

- shared tasks
- today and tomorrow preparation
- appointments
- insurance verification
- balances and collections
- comments and contact logs
- manager reporting
- admin operations

## Logging in / selecting a user

1. Open the desktop app.
2. Select your office user from the login list.
3. Click continue.

## Overdue and escalated items

When an item is overdue:

1. open it immediately
2. review notes and history
3. document contact attempts
4. claim or assign the work
5. escalate to manager when needed

## Common daily workflows

### Front desk opening routine

1. log in
2. open Today Board
3. work unconfirmed appointments
4. clear insurance pending items
5. review balance-due patients

### Midday routine

1. refresh boards
2. update comments/contact logs
3. claim urgent tasks
4. clear overdue items

### End-of-day routine

1. review Tomorrow Prep
2. finish confirmations
3. document unresolved issues
4. hand off escalations to manager

## Troubleshooting

### I cannot reach the server

- confirm the API host is running
- check that your workstation points to the correct API URL
- contact the office manager or IT support if the issue continues

### I see permission denied

- make sure you selected the correct user
- some screens require Manager or Admin access

### My task disappeared

- refresh the screen
- check filters
- review task history or ask a manager to confirm status changes
