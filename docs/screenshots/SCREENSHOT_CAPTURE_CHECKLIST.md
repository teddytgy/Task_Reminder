- `docs/screenshots/login-window.png`
- `docs/screenshots/main-dashboard.png`
- `docs/screenshots/today-board.png`
- `docs/screenshots/tomorrow-prep-board.png`
- `docs/screenshots/appointments-board.png`
- `docs/screenshots/insurance-queue.png`

- `docs/screenshots/manager-dashboard.png`
- `docs/screenshots/admin-operations.png`
- `docs/screenshots/notifications-settings.png`

Capture guidance:

1. Run the API with seeded demo data.
2. Run the WPF app and log in as the listed training user.
3. Resize the window to a readable training-friendly layout.
4. Capture one clean screenshot per screen with patient/sample data visible.
5. Save using the exact file names above so the manual can be updated without rewriting section names.

Suggested training users:

- Front desk workflow: `Mia Front Desk`
- Manager workflow: `Emma Insurance Manager`
- Admin workflow: `Ava Office Admin`
